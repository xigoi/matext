maTeXt

maTeXt is a program and Nim library for rendering LaTeX math as multiline Unicode text.

Unlike other LaTeX renderers, the output of maTeXt can be used in any medium that permits blocks of monospace text, including e-mails, Gemtext, etc. Using the one-line mode, it can even be used in environments that don't support preformatted blocks.

Installation

You need to have Nim and Nimble installed.

From Nimble

nimble install matext

From source

git clone https://git.sr.ht/~xigoi/matext
cd matext
nimble install

Usage

CLI

Supported features

The goal is to mostly match KaTeX's list of supported functions. Features which are not planned to be supported are marked with a cross.

  • digits, English letters
  • accents
    • simple accents
    • long accents
  • delimiters
    • most important delimiters
    • less important delimiters
    • \left \right
    • \big, etc.
  • environments
    • array, matrix
    • cases
    • equation, align, gather
    • CD
  • Greek and other letters
  • layout
    • \cancel, \sout
    • \tag
    • \overbrace, \underbrace
    • \not
    • \boxed
    • line breaks
    • superscript, subscript
    • \stackrel, \overset, \underset, \atop
  • spaces
  • macros
  • operators
    • big operators
    • binary operators
    • logic & set theory
    • \frac, \binom
    • \over, \choose
    • text operators
    • square root
    • n-th root
    • relations
    • negated relations
    • simple arrows
    • extensible arrows
    • bra-ket notation
  • color
  • fonts (only for one letter)
  • font size
  • display style
  • symbols
  • text mode
  • punctuation
  • units
